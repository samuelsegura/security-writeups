# Security Write-ups / Writeups de sécurité

## Français

Ces write-ups ont été créés à la base comme support 
personnel — un référentiel pour retrouver rapidement 
comment j'ai abordé une vulnérabilité spécifique sans 
avoir à refaire tout le lab.

Je résous chaque lab/CTF moi-même, puis j'utilise l'IA 
pour structurer la documentation. Cette approche me 
permet de me concentrer sur la pratique tout en gardant 
une trace exploitable de ma progression.

Je les partage ici pour transparence sur mon parcours 
d'apprentissage.
Chaque write-up est relu et validé avant d'être archivé.

Le template utilisé est disponible ici : [template_writeup.txt](./template_writeup.txt)

## English

These write-ups were originally created as a personal 
reference — a way to quickly revisit how I approached 
a specific vulnerability without redoing the entire lab.

I solve each lab/CTF myself, then use AI to structure 
the documentation. This approach lets me focus on the 
actual practice while keeping a usable record of my 
progress.

Sharing them here for transparency on my learning journey.
Each write-up is reviewed and validated before being archived.

The template used is available here: [template_writeup.txt](./template_writeup.txt)
